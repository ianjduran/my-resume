Dog Feeder

A Dog Feeder using a NodeMCU that is capable of reading a NFC Tag to recognize a pet and give the appropiate amount of food. In case the dog wants more food it needs to wait for some time, previously set by the pet owner.

Each Request is able to register the attempt into a database by using HTTP requests and SQL to manage the queries. Also a Telegram bot is able to give useful information about the pet to the owner.

  • Technologies Used: NodeJS, SQL, Arduino, Postman, Python
